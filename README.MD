# Job Tracking

Job Tracking adalah sebuah projek yang dibuat untuk memantau progress lamaran kerja yang udah saya apply. Projek ini dibuat dengan menggunakan bahasa pemrograman PHP dan framework Laravel. Projek ini juga sebagai portofolio laravel saya

## Fitur

- Memantau progress lamaran kerja yang sudah di apply
- Menggunakan sistem authentikasi yang aman
- Menampilkan chart berapa yang dighosting dan sampai interview

## Cara Penggunaan

1. Clone repository ini ke dalam direktori lokal Anda.
2. Jalankan perintah `composer install` untuk menginstalasi dependensi yang dibutuhkan.
3. Buatkan file `.env` dan isi dengan konfigurasi database Anda.
4. Jalankan perintah `php artisan migrate` untuk membuat tabel yang dibutuhkan.
5. Jalankan perintah `php artisan serve` untuk menjalankan aplikasi ini.
6. Buka browser Anda dan akses ke alamat `http://localhost:8000` untuk menggunakan aplikasi ini.
